## **2. Communication Standards (The "Language")**
To share intelligence automatically across different platforms, specific open standards are used.

> [!abstract] **The Standards Trio**
> *   **ğŸ“¦ STIX (Structured Threat Information Expression):**
>     The **Specification**. It defines *what* the threat info looks like (the file format). Includes CybOX.
> *   **ğŸšš TAXII (Trusted Automated Exchange of Indicator Information):**
>     The **Transport**. It defines *how* to send the info (Application layer protocol over HTTPS). Designed to carry STIX.
> *   **ğŸ”¬ CybOX (Cyber Observable Expression):**
>     The **Schema**. Standardizes how to describe events/properties (e.g., a file hash, an IP, a registry key). Now incorporated into STIX.

## **3. Threat Intelligence Platforms (TIP)**

**Problem:** Too many data sources with different formats.
**Solution:** **TIP** centralizes and aggregates data into a readable format.

**Three Main Data Types Aggregated:**
1.  **IOCs:** Indicators of Compromise.
2.  **TTPs:** Tools, Techniques, and Procedures.
3.  **Reputation:** Scores for domains/IPs.
